Detailed Description

The RegistrationController class Contains endpoints for the MembershipManagementMicroservice API.

This class can create, retrieve, update user data.

<response code="200">Returns the newly created user</response> <response code="400">If the item is null</response> <response code="500">If there was an internal server error</response>
